# SAMACK SHORTENER

Transformando URLs em links inteligentes de forma simples, rápida e 100% gratuita.

## 🚀 Recursos
- **Totalmente Estático**: Hospedado de forma nativa e rápida no GitHub Pages.
- **Configuração de Banco de Dados**: Conecte ao seu próprio Supabase de forma dinâmica na interface (ou use o armazenamento local offline).
- **Segurança**: Opções de proteção por senha e data de expiração para os links.
- **Estatísticas Avançadas**: Gráficos de cliques e analytics detalhados por dispositivo, navegador e localidade.
- **QR Code Inteligente**: Geração automática de QR code para cada link encurtado.

## 🛠️ Desenvolvimento Local
Para iniciar o projeto em sua máquina:
1. Instale as dependências com `npm install`.
2. Rode o servidor local com `npm run dev` ou use o script `iniciar.bat`.
3. Acesse `http://localhost:5173/samacklinks/`.

## 📦 Publicação
Dê um duplo clique no executável `deploy_samack.exe` na raiz do projeto para subir as atualizações automaticamente para o GitHub.
